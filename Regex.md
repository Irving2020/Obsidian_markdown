基本用法：
	/123 : 模糊匹配123456
	/^123$   :  严格匹配123
	[ ] : 匹配其中一个字符即可
	[ ^ ] : 取反
	[ ]{ 3, } : 匹配字符长度为3 , ∞
	.  : 匹配不包括回车符的任意字符

特殊字符：

'\d' : equal to [ 0 - 9 ]
'\D': equal to [ ^ 0 - 9 ] : 除 0 - 9 外任意字符
'\s': equal to [ 回车符 \r \n Tab符 \t  翻页符 \f \v  ]


group:（）匹配获取括号内匹配的内容
\1 : 引用组1
\k<name> : 引用组name

eg:（?<name >[ a-zA-Z0-9 ]） 组命名

eg: foo(?=bar) 匹配必须出现在bar前面的foo
